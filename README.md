🍳 Recipe Sharing App
The Recipe Sharing App is a full-stack web application where users can discover, share, and review recipes. The platform allows home cooks and food enthusiasts to upload recipes, interact with other users through comments and ratings, and save their favorite recipes.

The system also includes an admin moderation workflow where recipes must be approved before becoming publicly visible.

The application is built using modern web technologies with a React frontend and Node.js backend, ensuring a scalable and responsive user experience.
